# Telco Customer Churn Prediction

Задача бинарной классификации по прогнозированию оттока клиентов на основе табличных данных.

---

## Обзор проекта

Проект посвящён решению задачи customer churn prediction с использованием классического
пайплайна машинного обучения. Основной акцент сделан на работе с табличными данными,
feature engineering и корректной валидации модели.

Цель проекта - продемонстрировать подход к решению ML-задач, структуру кода
и понимание полного цикла построения модели.

---

## ML-пайплайн

В проекте реализованы следующие этапы:

- загрузка и предварительная обработка данных
- разведочный анализ данных (EDA)
- feature engineering
- обучение моделей
- оценка качества и валидация

---

## Модели и инструменты

- Тип задачи: бинарная классификация
- Используемые библиотеки:
  - pandas
  - NumPy
  - matplotlib
  - scikit-learn
  - catboost

---

## Результаты

Качество моделей оценивалось с использованием стандартных метрик бинарной классификации
(accuracy, precision, recall, ROC-AUC).

---

## Структура проекта

- `telco.py` — полный end-to-end ML-пайплайн:
  загрузка данных, EDA, feature engineering, обучение и оценка модели
